# Интернет-магазин Willberries

RESTful приложение "Willberries". Фронтенд реализован с помощью React JS, а API для сайта написан на PHP.
Ссылка на проект с API: https://github.com/iaminsleep/Willberries-API.

### Технологии:

- reactjs
- css
- javascript
- rest api

### Библиотеки:

- Bootstrap v5
- Swiper Slider JS
